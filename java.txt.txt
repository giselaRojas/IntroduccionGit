¿Qué es java?
Java es un lenguaje de programación ampliamente utilizado para codificar aplicaciones web. Ha sido una opción popular entre los desarrolladores durante más de dos décadas, con millones de aplicaciones Java en uso en la actualidad

¿Qué se puede hacer con Java?
Con Java puedes programar prácticamente cualquier cosa, es un lenguaje de programación muy potente, estable, rápido que brinda mucha seguridad a los programas.

Java es un lenguaje de propósito general, esto quiere decir que puedes programar prácticamente cualquier cosa que se te ocurra como:

Videojuegos
Aplicaciones móviles para Android
Aplicaciones de escritorio
Aplicaciones web
Aplicaciones para Smart TV
Apps para sistemas de navegación en automóviles
¡Y muchas otras cosas!

Es importante mencionar que Java no se ejecuta en todos los dispositivos, porque es un lenguaje que genera un código para un software especial que se llama “Máquina Virtual de Java” (JVM). El dispositivo donde quieres ejecutar tu programa debe tener instalada esta máquina virtual; pero no te preocupes, esta es compatible con los sistemas operativos más usados: Windows, Linux y macOS.
